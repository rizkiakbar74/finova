# Ringkasan Project Finova

Finova adalah aplikasi keuangan pribadi untuk pencatatan manual. Aplikasi membantu pengguna memantau pemasukan, pengeluaran, anggaran, tujuan tabungan, laporan, tagihan rutin, dan langganan.

## Teknologi

- Next.js App Router.
- TypeScript.
- Tailwind CSS.
- Supabase Auth.
- Supabase Database dengan Row Level Security.
- Supabase Storage private untuk receipt.

## Fokus Produk

- Pengalaman ringan dan mudah dipakai.
- Mata uang IDR.
- Antarmuka Bahasa Indonesia.
- Tidak memakai data dummy pada UI production.
- Keamanan data pengguna menjadi prioritas utama.
