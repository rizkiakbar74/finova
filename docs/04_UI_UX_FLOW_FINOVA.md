# Alur UI/UX Finova

## Alur Publik

```text
Beranda
-> Daftar
-> Onboarding
-> Dashboard
```

## Alur Pengguna

```text
Dashboard
-> Dompet
-> Kategori
-> Tambah Transaksi
-> Anggaran
-> Tujuan Tabungan
-> Laporan
-> Pengaturan
```

## Prinsip Tampilan

- Halaman pertama harus langsung memberi akses ke pengalaman utama.
- Tombol penting jelas dan mudah ditekan.
- Mobile bottom navigation tidak boleh menutup konten penting.
- Loading state harus berada di tengah layar dan terasa elegan.
- Teks memakai Bahasa Indonesia yang ringkas.

## Empty State

Empty state harus menjelaskan tindakan berikutnya, misalnya membuat dompet, membuat transaksi, atau mengubah filter.
